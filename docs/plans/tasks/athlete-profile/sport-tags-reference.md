# Sport Tags Reference

All pre-canned sports with their positional groups and tags. This is the seed
data for supported sports. Users can edit these tags or create entirely new
sports with custom tags.

**Supported sports (ship with pre-canned tags):** Soccer, Flag Football,
American Football, Basketball, Lacrosse, Rugby

---

## Soccer

| Position | Tags |
|----------|------|
| Attacker | Goal, Assist, Dribble |
| Midfielder | Pass, Chance Creation, Control |
| Defender | Tackle, Interception, Build-Up |
| Goalie | Save, Distribution |

### Tag Descriptions

- **Goal** — All types of finishes in scoring situations.
- **Assist** — Final pass that leads directly to a goal.
- **Dribble** — Beating defenders in individual attacking duels.
- **Pass** — Short, long, and line-breaking passes to advance play.
- **Chance Creation** — Key passes that set up opportunities for teammates.
- **Control** — Quick turnover play and composure under pressure.
- **Tackle** — Clean defensive challenges that win the ball.
- **Interception** — Reading and winning possession before the opponent.
- **Build-Up** — Accurate passes from the back that start attacks.
- **Save** — All shot-stopping and 1v1 saves.
- **Distribution** — Accurate throws and kicks to start counterattacks.

## Flag Football

| Position | Tags |
|----------|------|
| Quarterback | Touchdown Pass, Scramble, Play Action |
| Receiver | Touchdown Catch, Route Running, YAC |
| Center | Snap, Block, Route |
| Rusher | Sack, Pressure, Flag Pull |
| Defensive Back | Interception, Pass Breakup, Flag Pull |

### Tag Descriptions

- **Touchdown Pass** — Completed pass that results in a score.
- **Scramble** — Evading rushers to extend the play or gain yards.
- **Play Action** — Fake handoff that freezes the defense before a pass.
- **Touchdown Catch** — Receiving a pass in the end zone for a score.
- **Route Running** — Crisp cuts and separation from defenders on passing routes.
- **YAC** — Yards after catch; gaining extra ground after the reception.
- **Snap** — Clean snaps that start the play on time.
- **Block** — Positioning to shield the quarterback from rushers.
- **Route** — Center releasing into a passing route after the snap.
- **Sack** — Pulling the quarterback's flag behind the line of scrimmage.
- **Pressure** — Forcing the quarterback into a hurried throw or scramble.
- **Flag Pull** — Pulling the ball carrier's flag to end the play.
- **Interception** — Catching a pass intended for the offense.
- **Pass Breakup** — Deflecting or disrupting a pass without intercepting.

## American Football

| Position | Tags |
|----------|------|
| Quarterback | Touchdown Pass, Deep Ball, Scramble |
| Wide Receiver / Tight End | Touchdown Catch, Route Running, YAC |
| Running Back | Touchdown Run, Broken Tackle, Receiving |
| Offensive Line | Pancake Block, Pass Protection |
| Defensive Line | Sack, TFL, Run Stop |
| Linebacker | Tackle, Blitz, Coverage |
| Defensive Back | Interception, Pass Breakup, Tackle |
| Kicker / Punter | Field Goal, Punt, Kickoff |

### Tag Descriptions

- **Touchdown Pass** — Completed pass that results in a score.
- **Deep Ball** — Accurate throws of 20+ yards downfield.
- **Scramble** — Evading pressure to extend the play or gain yards.
- **Touchdown Catch** — Receiving a pass in the end zone for a score.
- **Route Running** — Crisp cuts and separation from defenders on passing routes.
- **YAC** — Yards after catch; gaining extra ground after the reception.
- **Touchdown Run** — Rushing the ball into the end zone for a score.
- **Broken Tackle** — Breaking through or spinning off a defender's tackle attempt.
- **Receiving** — Catching passes out of the backfield.
- **Pancake Block** — Driving a defender flat to the ground.
- **Pass Protection** — Keeping the pocket clean for the quarterback.
- **Sack** — Tackling the quarterback behind the line of scrimmage.
- **TFL** — Tackle for loss; stopping the ball carrier behind the line.
- **Run Stop** — Stuffing a run play at or near the line of scrimmage.
- **Tackle** — Bringing down the ball carrier in the open field.
- **Blitz** — Rushing the quarterback from a linebacker position.
- **Coverage** — Defending a pass from a linebacker position.
- **Interception** — Catching a pass intended for the offense.
- **Pass Breakup** — Deflecting or disrupting a pass without intercepting.
- **Field Goal** — Successful kick through the uprights for 3 points.
- **Punt** — Booming kicks that pin the opponent deep.
- **Kickoff** — Strong kicks that limit return yardage.

## Basketball

| Position | Tags |
|----------|------|
| Guard | Scoring, Assist, Steal |
| Forward | Scoring, Rebound, Block |
| Center | Post Move, Rebound, Block |

### Tag Descriptions

- **Scoring** — Baskets from any range: layups, mid-range, three-pointers, dunks.
- **Assist** — Pass that directly leads to a teammate's basket.
- **Steal** — Taking the ball from the opposing team.
- **Rebound** — Grabbing the ball after a missed shot.
- **Block** — Deflecting or swatting an opponent's shot attempt.
- **Post Move** — Skilled footwork and finishing in the low post area.

## Lacrosse

| Position | Tags |
|----------|------|
| Attack | Goal, Assist, Dodge |
| Midfield | Ground Ball, Transition, Shot |
| Defense | Check, Ground Ball, Clear |
| Goalie | Save, Clear, Outlet |

### Tag Descriptions

- **Goal** — Scoring by putting the ball in the net.
- **Assist** — Pass that directly leads to a teammate's goal.
- **Dodge** — Beating a defender one-on-one to create a scoring chance.
- **Ground Ball** — Winning possession of a loose ball.
- **Transition** — Carrying or passing the ball quickly up the field on a fast break.
- **Shot** — On-target shots that test the goalkeeper.
- **Check** — Legal stick check that dislodges the ball from an opponent.
- **Clear** — Moving the ball from the defensive end past midfield.
- **Save** — Goalkeeper stopping a shot.
- **Outlet** — Quick pass after a save to start the transition.

## Rugby

| Position | Tags |
|----------|------|
| Forward | Tackle, Carry, Ruck |
| Back | Try, Line Break, Kick |
| Halfback | Pass, Box Kick, Snipe |

### Tag Descriptions

- **Tackle** — Bringing down the ball carrier and competing for the ball.
- **Carry** — Running with the ball into contact, gaining ground.
- **Ruck** — Securing or contesting possession at the breakdown.
- **Try** — Scoring by grounding the ball in the in-goal area.
- **Line Break** — Breaking through the defensive line into open space.
- **Kick** — Tactical kicking: conversions, penalties, territory kicks.
- **Pass** — Accurate service from the ruck to the backline.
- **Box Kick** — Contestable kick from behind the ruck.
- **Snipe** — Darting run close to the ruck to exploit gaps.
